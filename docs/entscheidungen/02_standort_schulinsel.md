# Entscheidungen: Standort für das Projekt

- **Maßnahme:** Verlagerung der geplanten Pfandsammelstation in die internen Räumlichkeiten.
- **Begründung:** Im Außenbereich fehlen Aufsichtsmöglichkeiten, zudem besteht die Gefahr von Fremdeingriffen (z. B. Verlust von Sammelgut). Innenräume bieten bessere Kontrolle und Erreichbarkeit.
- **Ziel:** Sicherung des Projekterfolgs durch eine kontrollierte Umgebung.
- **Prüfhinweis / Beobachtungskriterium:** Wird die interne Sammelstation von X und anderen genutzt, oder sinkt die Motivation durch die veränderte Positionierung?
