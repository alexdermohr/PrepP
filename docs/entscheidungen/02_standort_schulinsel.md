# Entscheidungen: Standort für das Projekt

- **Maßnahme:** Verlagerung der geplanten Pfandsammelstation von einer externen Mülltonne in die internen Räumlichkeiten.
- **Begründung:** Bedenken hinsichtlich Vandalismus, Sicherheit des Inhalts und fehlender Aufsicht im Außenbereich. Innenräume bieten Schutz, Aufsicht und gute Erreichbarkeit.
- **Ziel:** Schaffung einer realistischen, sicheren Umgebung für das Projekt.
