# Mapping: Gruppennachweis → Hausarbeit

## Zweck

Dieses Dokument stellt sicher, dass jede Aussage in der Hausarbeit auf eine Quelle im Gruppennachweis (oder darüber hinaus auf Rohmaterial) zurückgeführt werden kann.

**Regel:** KEINE Aussage ohne Rückverweis.
