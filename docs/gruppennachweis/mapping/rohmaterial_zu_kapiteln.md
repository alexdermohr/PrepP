# Mapping: Rohmaterial → Gruppennachweis-Kapitel

## Zweck

Dieses Dokument stellt sicher, dass jede Aussage im Gruppennachweis auf eine Quelle im Rohmaterial zurückgeführt werden kann.

**Regel:** KEINE Aussage ohne Rückverweis.
