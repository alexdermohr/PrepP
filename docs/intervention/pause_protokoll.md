# Pause-Protokoll (Minimal)

## Satz
„Ich mache kurz Pause.“

## Regel
- Jeder darf jederzeit „Pause“ sagen
- Dauer: 2 Minuten
- Danach Rückkehr

## Kärtchen
Optional: sichtbares „Pause“-Kärtchen

## Training
3 Durchläufe:
- Situation → Pause → zurückkommen

## Rückkehrsatz
„Ich will X erreichen. Mein Punkt ist Y.“

## Done
- Satz bekannt
- Regel akzeptiert
- 3× geübt
